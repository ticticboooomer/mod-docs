# Duration

This is the processing length required to complete the process, this is measured in ticks, there is 20 ticks per second

so a duration of... 

* 20 = 1 second
* 100 = 5 seconds
* 200 = 10 seconds
* 400 = 20 seconds
* 1200 = 1 minute
* 6000 = 5 minutes

Example
```json
"duration": 400
```
