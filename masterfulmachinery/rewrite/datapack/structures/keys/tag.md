# Structure Part - Block Tag

The `mm:block` structure part is simply a key for a specific block in a structure.

### The Structure Part Id
```json
{
    "type": "mm:tag",
    ...
}
```

### Full Definition
```json
{
    "type": "mm:tag",
    "tag": "<block tag id>"
}
```

### Full Example
```json
{
    "type": "mm:tag",
    "tag": "minecraft:wool"
}
```
