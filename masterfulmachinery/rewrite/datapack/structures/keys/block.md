# Structure Part - Block

The `mm:block` structure part is simply a key for a specific block in a structure.

### The Structure Part Id
```json
{
    "type": "mm:block",
    ...
}
```

### Full Definition
```json
{
    "type": "mm:block",
    "block": "<block id>"
}
```

### Full Example
```json
{
    "type": "mm:block",
    "block": "minecraft:glass"
}
```
