# Name

This can be either `text` or a `translation` key

### Text
```json
  "name": {
    "text": "Giant Item Input"
  },
```

### Translation
```json
    "translation": "gui.mm.solar_panel_item_input_port"
```

| :memo:  | `Translation` replaces the entire `"name"` object |
|---------|:--------------------------------------------------|
