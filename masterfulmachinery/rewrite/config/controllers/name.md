# Name

this can be either `text` or a `translation` key

### Text
```json
  "name": {
    "text": "Cool Controller Name"
  },
```

### Translation
```json
    "translation": "gui.mm.solar_panel_controller"
```

| :memo:  | `Translation` replaces the entire `"name"` object |
|---------|:--------------------------------------------------|
